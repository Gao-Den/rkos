Task

Operating System task management, scheduling, synchronization and inter-task communication services.

Task management

void task_os_init()

Initialize the task management system: Memory pool, Message package, Heap system, Timer service.

Note

MUST-BE called before any other task os function

Parameters:

none – [in]

Returns:

none

void task_os_create(uint8_t task_id, pf_task task_handler, uint8_t task_priority, uint32_t task_stack_size, void *mailbox, uint16_t mailbox_size, const uint8_t *task_name)

Create a single task.

Note

MUST-BE called after task_os_init and before task_os_run

Parameters:

  • task_id – [in] task id (MUST-BE unique and less than TASK_POOL_MAX_SIZE)

  • task_handler – [in] task handler function pointer

  • task_priority – [in] task priority (reference task_priority_t)

  • task_stack_size – [in] task stack size (unit: uint32_t, MUST-BE greater than 0)

  • mailbox – [in] pointer to the mailbox object (set MAILBOX_NULL if the task does not require a mailbox)

  • mailbox_size – [in] mailbox size (set the value to 0 if the task does not require a mailbox)

  • task_name – [in] pointer to the task name string

Returns:

none

void task_post_common_msg(uint8_t des_task_id, uint8_t signal, uint8_t *data, uint32_t size)

Task post a common message.

Parameters:

  • signal – [in] message execution signal

tcb_t *task_list_get(task_list_t *list)

Remove and return the head task control block from the given task list.

Warning

kernel private (should not be called by user applications directly)

Parameters:

list – [in] pointer to the task list (MUST-BE not NULL)

Returns:

pointer to the task control block of the head task list. If the list is empty, it will return NULL

tcb_t *task_list_get_head(task_list_t *list)

Return the head task control block from the given task list without removing it.

Warning

kernel private (should not be called by user applications directly)

Parameters:

list – [in] pointer to the task list (MUST-BE not NULL)

Returns:

pointer to the task control block of the head task (not removed from the list). If the list is empty, it will return NULL

tcb_t *task_list_get_higher_priority(task_list_t *list)

Parameters:

Returns:

pointer to the task control block of the highest priority task (removed from the list). If the list is empty, it will return NULL

bool task_list_is_available(task_list_t *list)
