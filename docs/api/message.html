Message

Message package management provides memory pool allocation and APIs for creating, accessing and releasing pure, common and dynamic messages used for inter-task communication.

Initialization

void msg_init()

Message initialization (Message pool)

Note

This function will be called by the kernel

Parameters:

none – [in]

Returns:

none

Message memory management

void free_msg(rk_msg_t *msg)

Free a message (Any type of message)

Parameters:

msg – [in] pointer to the message to be freed (MUST-BE not NULL)

Returns:

none

none – [in]

Returns:

pointer to the allocated pure message

uint8_t get_pure_msg_free()

Get a available pure message (Message free in the pool)

none – [in]

void set_data_common_msg(rk_msg_t *msg, uint8_t *data, uint8_t size)

Set data for a common message.

Parameters:

none
